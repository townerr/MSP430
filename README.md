# MSP430G2253

Projects to play around with MSP430

### 1. Blink1
    - Blinking both green and red LEDS.
### 2. Blink2
    - Blinking both LEDS but more simplified.
### 3. ButtonLEDOff
    - Use push button to turn off LED.
### 4. ButtonBlinkNTimes
    - Press button to blink the LED N times. Default is 5.
